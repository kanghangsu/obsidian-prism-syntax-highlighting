# Obsidian Prism Syntax Highlighting

Obsidian의 기본 구문 강조 기능을 개선하는 플러그인으로, 편집 모드와 읽기 모드 모두에서 일관되고 강화된 코드 구문 강조를 제공합니다.

## 문제 배경

Obsidian의 기본 Prism 구현은 CodeMirror 6 편집 모드에서 토큰화를 제대로 처리하지 못하는 문제가 있습니다. 이로 인해 편집 모드와 읽기 모드 간에 구문 강조가 일관되지 않고, 일부 언어에서 구문 강조가 제대로 작동하지 않습니다.

## 주요 기능

- 개선된 구문 강조: 코드 블록에 더 정확하고 일관된 구문 강조 제공
- 이중 모드 지원: 읽기 모드와 편집 모드 모두에서 동일한 구문 강조 경험
- 200+ 언어 지원: 대부분의 인기 프로그래밍 언어 지원
- 동적 로딩: 필요한 언어만 필요할 때 로드하여 성능 최적화
- 실시간 업데이트: 편집 중에도 즉시 구문 강조 반영

## 설치

### 수동 설치
1. 이 저장소를 다운로드
2. `.obsidian/plugins/obsidian-prism-syntax-highlighting/` 디렉토리에 압축 해제
3. Obsidian을 재시작하고 설정에서 플러그인 활성화

## 사용 방법

설치 후 자동으로 작동합니다. 코드 블록에 언어를 지정하면 해당 언어에 맞는 구문 강조가 적용됩니다.
